# Racewalk Tournament

A real-time scoring and event management system for Racewalk competitions. Features a live leaderboard, multi-judge card issuance, and cloud-based officiating.

Developed by **Solinex**

---

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript** (strict mode)
- **Tailwind CSS v4** + shadcn/ui
- **Prisma 7** with MySQL / MariaDB
- **NextAuth 4** (JWT strategy)

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Configure environment variables

Copy `.env.example` to `.env` and fill in the required values:

```bash
cp .env.example .env
```

| Variable | Example |
|----------|---------|
| `DATABASE_URL` | `mysql://user:password@host:3306/database` |
| `NEXTAUTH_SECRET` | random secret string |
| `NEXTAUTH_URL` | `http://localhost:3000` |

### 3. Sync the database

```bash
npm run db:push
```

### 4. Seed admin users

```bash
npm run db:seed
```

### 5. Start the dev server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Commands

```bash
npm run dev           # Start dev server
npm run build         # Production build
npm run lint          # ESLint check
npm run db:push       # Sync Prisma schema to DB
npm run db:seed       # Seed admin users (idempotent)
npm run db:seed:reset # Wipe auth tables and reseed
```

## Role Structure

| Role | Route |
|------|-------|
| Admin | `/admin/` |
| Judge | `/judge/events/[eventId]/` |
| Head Judge | `/head-judge/events/[eventId]/` |
| Event Logger | `/event-logger/events/[eventId]/` |
| Timekeeper | `/timekeeper/events/[eventId]/` |
| Public Scoreboard | `/events/[eventId]/` |

Each non-admin user joins their workspace using a **6-character secret code** generated by the Admin per round.
